# 网络访问指南

本项目是一个纯前端的HTML应用，可以通过多种方式实现网络访问。

## 方案一：局域网访问（最简单）

### 1. 启动本地服务器

**使用Python（推荐）：**
```bash
# 在项目目录下执行
python -m http.server 8000
```

**使用Node.js（如果已安装）：**
```bash
# 安装http-server（首次使用）
npm install -g http-server

# 启动服务器
http-server -p 8000
```

**使用PHP（如果已安装）：**
```bash
php -S 0.0.0.0:8000
```

### 2. 获取本机IP地址

**Windows:**
```bash
ipconfig
# 查找 IPv4 地址，例如：192.168.1.227
```

**Mac/Linux:**
```bash
ifconfig
# 或
ip addr show
```

### 3. 访问链接

- **本机访问：** http://localhost:8000
- **局域网访问：** http://你的IP地址:8000
  - 例如：http://192.168.1.227:8000

### 4. 配置防火墙（Windows）

1. 打开"Windows Defender 防火墙"
2. 点击"高级设置"
3. 选择"入站规则" → "新建规则"
4. 选择"端口" → TCP → 特定本地端口：8000
5. 允许连接 → 完成

## 方案二：内网穿透（外网访问）

### 使用 ngrok（推荐，免费）

1. **注册并下载ngrok**
   - 访问：https://ngrok.com/
   - 注册账号并下载ngrok

2. **配置authtoken**
   ```bash
   ngrok config add-authtoken 你的token
   ```

3. **启动本地服务器**
   ```bash
   python -m http.server 8000
   ```

4. **启动ngrok**
   ```bash
   ngrok http 8000
   ```

5. **获取公网链接**
   - ngrok会提供一个类似 `https://xxxx.ngrok.io` 的链接
   - 这个链接可以在任何地方访问

### 使用 frp（自建服务器）

需要有自己的服务器，配置较复杂，适合长期使用。

## 方案三：部署到云服务器

### 使用Nginx部署

1. **在服务器上安装Nginx**
   ```bash
   # Ubuntu/Debian
   sudo apt update
   sudo apt install nginx
   
   # CentOS
   sudo yum install nginx
   ```

2. **配置Nginx**
   ```nginx
   server {
       listen 80;
       server_name 你的域名或IP;
       
       location / {
           root /path/to/testc116;
           index index.html;
       }
   }
   ```

3. **上传文件到服务器**
   - 使用FTP、SCP或Git上传项目文件

4. **启动Nginx**
   ```bash
   sudo systemctl start nginx
   sudo systemctl enable nginx
   ```

## 方案四：静态网站托管（免费）

### GitHub Pages

1. **创建GitHub仓库**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/仓库名.git
   git push -u origin main
   ```

2. **启用GitHub Pages**
   - 进入仓库设置 → Pages
   - 选择分支和目录
   - 访问：https://你的用户名.github.io/仓库名/

### Vercel / Netlify

1. 注册账号
2. 导入GitHub仓库
3. 自动部署，获得免费域名

## 方案五：使用批处理脚本（Windows一键启动）

创建一个 `start-server.bat` 文件：

```batch
@echo off
echo 正在启动服务器...
echo.
echo 本机访问: http://localhost:8000
echo.
for /f "tokens=2 delims=:" %%a in ('ipconfig ^| findstr /i "IPv4"') do (
    set IP=%%a
    goto :found
)
:found
echo 局域网访问: http://%IP: =%:8000
echo.
echo 按 Ctrl+C 停止服务器
echo.
python -m http.server 8000
pause
```

## 推荐方案

- **临时测试/局域网使用：** 方案一（Python HTTP服务器）
- **外网临时访问：** 方案二（ngrok）
- **长期外网访问：** 方案三（云服务器）或方案四（静态托管）

---

## 注意事项

1. **安全性：** 纯前端应用，数据存储在浏览器localStorage中，相对安全
2. **性能：** 本地服务器适合小规模使用，大规模访问建议使用专业服务器
3. **端口：** 确保防火墙允许相应端口访问
4. **HTTPS：** 如需HTTPS，可以使用Let's Encrypt免费证书
